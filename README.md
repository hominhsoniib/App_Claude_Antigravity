## 📌 Giới thiệu

NEXUS CRM là nền tảng quản trị doanh nghiệp được phát triển nhằm số hóa toàn bộ quy trình kinh doanh từ tiếp nhận khách hàng đến báo giá, hợp đồng, chăm sóc khách hàng và quản lý giấy phép phần mềm.

Hệ thống được xây dựng theo kiến trúc module giúp mở rộng dễ dàng và tích hợp AI trong từng nghiệp vụ.

# ⚙️ Cài đặt

Clone Project

```bash
git clone git@github.com:hominhsoniib/App_Claude_Antigravity.git
```

Di chuyển

```bash
cd App_Claude_Antigravity
```

Tạo môi trường

```bash
python -m venv venv
```

Kích hoạt

Windows

```bash
venv\Scripts\activate
```

Linux

```bash
source venv/bin/activate
```

Cài thư viện

```bash
pip install -r requirements.txt
```
